Pyoko Pyoko Ultra! / Chick Platformer
v1.0
Created 8/14

Description: Dodge the enemies and eat your way to victory! Help the chick reach the end!

Currently developing mechanics, in very early stages but already has the bases to make simple levels. Two experimental levels
are in place (to be fixed). No enemy AI yet or sprites for everything. Current animations for the main player is in place.
Sound to be added. 

Skills: Unity- 2d physics implementation (not default unity physics), camera, gizmos, moving platforms, animation, crouching, 
sprinting, jumping, game design   
C#- class structure
